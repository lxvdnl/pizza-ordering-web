# How to install and run
1. Clone the repository
``` 
git clone https://github.com/lxvdnl/pizza-ordering-web.git 
```
2. Install node.js
3. Including dependencies
```
cd pizza-ordering-web
npm install
```
4. Start the server
```
node server.js
```
5. Now you can visit the site: http://localhost:3000
